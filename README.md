# tradingedge.club-scraper

### Prerequisites

#### Using Poetry

```sh
python -m venv .venv
# then you need to activate the virtual environment for your shell
pip install poetry
poetry install # switched to poetry for dependency management
```

#### Using pip

```sh
python -m venv .venv
pip install -r requirements.txt
```
